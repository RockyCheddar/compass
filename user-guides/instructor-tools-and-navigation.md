# Instructor Tools and Navigation

### Course Management

From the Course page, instructors can:

* Edit course details
* Manage Skills Assessment Mode
* View enrolled activities
* Access course statistics

### Learning Objects Management

Instructors with Content Creator privileges can:

* Create new Learning Objects
* Edit existing Learning Objects
* Link Learning Objects to Activities

### Case Studies Access

Instructors can:

* View Case Studies in Condensed View
* Review Case Study metadata
* See which Learning Objects use a particular Case Study

{% hint style="success" %}
**Note**: For creating and editing Learning Objects and Case Studies, instructors should refer to the [Course Designer Guide.](course-designer/course-designer-guide.md)
{% endhint %}
