# Creating Learning Objectives

What You’ll Learn

* How to create a new learning objective
* How to duplicate an existing learning objective
* Steps for filling in required information and linking a case study
* How to save your work

***

## Creating Learning Objects

There are two methods to create new Learning Objects:

### **Method 1: Create from scratch**

{% stepper %}
{% step %}
#### Make sure Designer Mode is enabled.

<div align="left"><figure><img src="../../.gitbook/assets/Create a New Learning Object in Course Designer - Step 2.png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Click the "Create new Learning Object" Magic-wand button in the top right

<div align="left"><figure><img src="../../.gitbook/assets/Create a New Learning Object in Course Designer - Step 3.png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Fill in the required information, including the name of the Learning Object and instructions for students.

<div align="left"><figure><img src="../../.gitbook/assets/Create a New Learning Object in Course Designer - Step 4.png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Select the Case Study you want to use.

<div align="left"><figure><img src="../../.gitbook/assets/Create a New Learning Object in Course Designer - Step 9.png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
#### Confirm by selecting ‘Save’.

<div align="left"><figure><img src="../../.gitbook/assets/Create a New Learning Object in Course Designer - Step 10.png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}
