# Editing Case Studies

This page explains how to edit case studies in EdEHR. It covers updating case details and editing EHR content.&#x20;

What You’ll Learn

* How to change case study properties like name and description
* How to edit the EHR data linked to a case study

***

## Editing Case Studies

Only Content Creator users can edit Case Study data:

**Editing Case Properties**:

1. From the Case Study page, use the edit button
2. You can modify:
   * Name: A short, descriptive title
   * Description: Information shared with other instructors
   * Version: Optional value to distinguish different versions
   * Contributors: Optional list of instructors who helped create this Case Study

**Editing EHR Content**:

1. From the Case Study page, click the "Edit in EHR" button
2. This launches the EHR with the case study data
3. You can modify form content and add assessments to tables
4. For other types of modifications, contact your EdEHR administrator

Step-by-Step Guide:

1. While editing a case study, sign in as the role you need (e.g., Nurse, Doctor).
2. Enter the relevant documentation (nursing notes, orders, etc.).
3. Sign out and back in as another role if you need to add documentation from a different perspective.



Sign in as the health care professional whose documentation you want to add or edit. For example, use a doctor or hospitalist account to enter doctor’s orders, and a nurse account for nursing documentation.
