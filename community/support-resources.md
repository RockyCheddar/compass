# Support Resources

Find help with EdEHR implementation, usage, and technical issues.

### Community Support

* **Slack Channel**: Ask questions in #technical-help
* **Discussion Forum**: Search for solutions or post new questions
* **Knowledge Base**: Browse our searchable documentation

### Technical Support

For official technical support:

* Submit a support ticket
* Email: support@edehr.org
* Response time: Typically within 1-2 business days

### Implementation Support

* Implementation guide
* Best practices documentation
* Schedule a consultation with our implementation team

### Bug Reporting

1. Check if the issue is already reported in our bug tracker
2. Gather information: browser/device, steps to reproduce, screenshots
3. Submit a detailed bug report

### Security Concerns

For security vulnerabilities, please report directly to security@edehr.org rather than posting publicly.
