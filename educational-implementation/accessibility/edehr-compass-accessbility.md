# edEHR Compass Accessbility

### GitBook Documentation Accessibility

Our EdEHR documentation is hosted on GitBook, which has its own accessibility features and considerations:

***

### GitBook Platform Features

* **Responsive Design**: GitBook documentation adapts to different screen sizes and devices.
* **Keyboard Navigation**: Users can navigate GitBook content using keyboard shortcuts.
* **Customizable Display**: GitBook offers dark/light mode options and adjustable text size.
* **Search Functionality**: Accessible search helps users quickly find specific information.

***

### Known GitBook Limitations

While GitBook provides many accessibility features, users should be aware of some potential limitations:

* Some complex interactive elements may have limited screen reader support
* Custom embedded content may have varying levels of accessibility
* Table navigation can be challenging with certain assistive technologies

***

### Tips for Accessing EdEHR Documentation

* Use GitBook's built-in search functionality to quickly locate specific information
* Leverage browser accessibility tools when needed (zoom, high contrast, etc.)
* For any documentation accessibility issues, please contact us using the information in the "Feedback" section

***

We continuously evaluate our documentation platform choices to ensure we provide the most accessible experience possible for all EdEHR users.
