## Accessibility Commitment

EdEHR is committed to ensuring the educational electronic health record system is accessible to all users, including those with disabilities. We believe that accessible design benefits everyone and is fundamental to creating an inclusive learning environment for healthcare education.

### Accessibility Audit

EdEHR has undergone a comprehensive accessibility audit conducted by the combined expertise of Easysurf and Untapped Accessibility. This professional assessment was designed to identify accessibility strengths and opportunities for improvement within our platform.

#### Audit Overview

The accessibility audit evaluated EdEHR against current accessibility standards, including:

* Web Content Accessibility Guidelines (WCAG) 2.1 Level AA compliance
* Keyboard navigation functionality
* Screen reader compatibility
* Color contrast and visual design
* Form and input accessibility
* Content structure and organization

The detailed findings of our accessibility audit will be available at <mark style="color:yellow;">\[Link to Findings Document]</mark>. These findings inform our ongoing accessibility improvements and development roadmap.

### Accessibility Features

EdEHR incorporates numerous accessibility features to support diverse user needs:

#### Keyboard Navigation

* Full keyboard accessibility for all platform functions
* Visible focus indicators for keyboard users
* Logical tab order matching visual layout

#### Screen Reader Compatibility

* Semantic HTML structure for proper screen reader interpretation
* Appropriate ARIA attributes where native HTML semantics are insufficient
* Meaningful alternative text for images and icons

#### Visual Design

* Sufficient color contrast ratios for text and interface elements
* Multiple visual cues beyond color alone (patterns, shapes, text labels)
* Responsive design for different screen sizes and zoom levels

#### Form Accessibility

* Clear, descriptive labels for all form fields
* Error identification and suggestions for correction
* Sufficient time to complete tasks without unexpected timeouts

### Accommodations for Students and Faculty

EdEHR supports various accommodations for students and faculty with disabilities:

* Extended time settings for timed activities
* Alternative formats for content where applicable
* Compatibility with common assistive technologies
* Flexible display settings for personalization

### Accessibility in Healthcare Education

Accessible educational tools like EdEHR help prepare healthcare students to:

* Understand diverse patient needs in clinical settings
* Develop inclusive communication practices
* Recognize accessibility considerations in healthcare delivery
* Build empathy for patients with different abilities

### Ongoing Accessibility Improvements

EdEHR views accessibility as an ongoing journey rather than a destination. We are committed to:

* Regular accessibility testing with each new release
* Prioritizing accessibility fixes in our development roadmap
* Collecting user feedback specific to accessibility experiences
* Staying current with evolving accessibility standards and best practices
